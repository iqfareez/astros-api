People In Space API

Welcome

Get to know who is currently in the space. Basic info is available such as their names, craft, and image. Feel free to use it in small/medium applications.

Base data are from Open Notify, images are from Bing Image Search API .

Web demo Android Demo

*Both demos are built using Flutter

Data health

Data will be updated from source every Monday and Thursday. Automated with GitHub actions.
